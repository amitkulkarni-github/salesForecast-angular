About this application

This application can be used for forecasting the sales in the upcoming years. User can provide certain inputs and the application forecasts the sales using the ARIMA model.

Parameter Description
History Start Year-Week and No. of Weeks This gives the range of the history data to be considered for the forecasting of sales
History Cut Off Date This is an informational field to tell the user about the day till history is available in the system
Forecast Start Year-Week and No. of Weeks This gives the range of the forecast data to be requested from the system
Channel The channels through which products are sold (such as online, in-store etc.)
Product The product(s) for which the forecast is requested
Forecast Method Different forecasting algorithms that fall under the ARIMA model. This field is optional
